# 📄 Resume Builder Web Application

A simple and interactive **Resume Builder Web App** that allows users to create professional resumes with a live preview and download them as a PDF.

---

## 🚀 Features

* ✨ Live resume preview while typing
* 🔄 Reset functionality to clear inputs
* 📄 Download resume as PDF
* 🎯 Clean and responsive UI
* 🧾 Supports multi-line input formatting (Education, Skills, Projects)

---

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and responsive layout
* **JavaScript** – DOM manipulation and interactivity
* **Libraries Used:**

  * `html2canvas` – Convert HTML to image
  * `jsPDF` – Generate and download PDF

---

## 📸 How It Works

1. User enters details in the form
2. Data updates instantly in the preview section
3. Preview is converted into an image
4. Image is added to a PDF
5. PDF is downloaded with one click
